## About My Approach

Every project in this portfolio follows the full data analytics lifecycle:

**Ask → Prepare → Process → Analyze → Share → Act**

Dashboards are built with a business stakeholder in mind — starting with the "so what?" before touching the data. I use the Guided Analytics Framework (Executive Summary → Focused Reports → Drill-through → Data Explorer) and the data storytelling arc to ensure every dashboard tells a clear, actionable story.
